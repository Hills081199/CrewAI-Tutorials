# CrewAI Tutorials

Tutorials for learning CrewAI, a framework for building multi-agent AI systems.

## Lessons

### Lesson 1: Create Agents to Research and Write an Article

Build a multi-agent system with Planner, Writer, and Editor agents that automatically researches and writes articles on any topic.
